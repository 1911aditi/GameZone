 **Ball in Maze** 

---

<br>

## **Description 📃**

- Ball in maze is a game to manipulating the maze board to get the ball towards the goal.

## **How to play? 🕹️**

- - In Ball-in-Maze, users can:
   - Start, pause, restart
   -  Move up, down, left, right
   - Restart the game if ball dropped
   - Modal for pause, rules.
